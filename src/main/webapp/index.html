RESTful para préstamos de equipos del LSI

/prestamos/api

  • /usuario
  • /codigo
  • /equipo
  • /prestamo
  1. El usuario debe crear un perfil en su aplicación de Android
  2. Antes de enviar los datos para su registro, debe escanear un codigo generado en la app de la tablet
  3. El codigo de la tablet se genera llamando por GET a http://lsi.no-ip.org:7777/prestamos/api/codigo/new
  4. Ese código debe usado en la URL cuando se da de alta el usuario por POST al http://lsi.no-ip.org:7777/prestamos/api/usuarios/edecaccaf0424ab1ae3c943d71cbe145 enviando el usuario como dato

      5. Respuestas posibles

    • El codigo ya ha sido validado anteriormente.
    • Usuario creado satisfactoriamente!
    • El código ingresado no existe.
  5. Una vez registrado, la app queda en modo lectura de codigo para escanear un código QR de un equipo
  6. El codigo escaneado es el ID y la descripción del equipo
  7. Una vez obtenido el id del equipo, se envia con el ID de usuario para generar el prestamo POST a http://localhost:8080/prestamos/api/prestamo enviando {"idAlumno":2,"idEquipo":4}
  8. Cuando se devuelve un equipo, se debe escanear el código en la tablet enviando el ID a por GET http://localhost:8080/prestamos/api/prestamo/{id}
  9. La tablet debe actualizar la lista de equipos prestados cada X tiempo

El metodo GET obtiene la lista de todos los elementos en la base http://lsi.no-ip.org:7777/prestamos/api/usuarios

El metodo GET con parametro obtiene el elemento con ese ID en la base http://lsi.no-ip.org:7777/prestamos/api/usuarios/{id}

El metodo GET /count obtiene la cantidad de todos los elementos en la base http://lsi.no-ip.org:7777/prestamos/api/usuarios/count

El metodo GET con dos parametros obtiene la lista de todos los elementos en la base con ID entre los valores dados http://lsi.no-ip.org:7777/prestamos/api/usuarios/{desde}/{hasta}

El metodo POST crea un elemento en la base. Se envía el objeto a crear.

El metodo PUT modifica un elemento en la base. Se envia el ID por parametro en la url /{id} y el objeto a modificar

El metodo DELETE elimina un elemento en la base. Se envia el ID por parametro en la url /{id}

Para crear un nuevo codigo usar el GET a http://lsi.no-ip.org:7777/prestamos/api/codigo/new

Para buscar un código usar el GET enviando el codigo http://lsi.no-ip.org:7777/prestamos/api/codigo/find/5b411a37f1d84ae3b635ab2940c3c6d5

Para registrar un nuevo usuario enviar el código por URL y los datos del usuario adjuntos por POST http://lsi.no-ip.org:7777/prestamos/api/usuario/6f270495e4754a6db5ae4b0c2fe2d556 {"legajo": "12322","nombre": "otro rest","rol": "ALUMNO","telefono": "3221"}
